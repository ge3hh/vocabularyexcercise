# Vocabulary Exercise

一个基于 Web 页面和 SQLite 的英语词汇练习工具。

当前版本支持：
- 艾宾浩斯式复习间隔
- 错题本
- 独立统计面板窗口
- 记录单词首次测试时间，并在统计窗口展示最近首次测试记录
- 多词库支持：高中词库、大学四级词库、大学六级词库

## 运行环境

- Python 3
- SQLite3

Web 版使用 Python 标准库实现，本地启动不依赖第三方 Web 框架。

## 启动程序

在项目目录下直接运行：

```bash
python web_app.py
```

默认会启动本地 Web 服务。

然后在浏览器打开：

```text
http://127.0.0.1:5000
```

只要机器上有 Python 环境和浏览器，就可以直接使用这个页面版本，不需要额外安装 Web 依赖。

如果需要让同一局域网内其他设备访问，可以把运行机器的 IP 地址加上端口 `5000` 打开，例如：

```text
http://192.168.1.10:5000
```

## 主界面说明

主界面包含以下主要区域：

- 词库与测试区：切换词库、导入导出、开始测试
- 当前单词卡片：显示当前测试的英文单词与释义
- 本轮已学习列表：显示这一轮已经答过的单词
- 本轮错题列表：显示当前这一轮答错的单词，并支持导出
- 统计与记录区：显示词库统计、首次测试记录、错题本

## 导入词库

程序提供两种导入方式。

### 方式 1：导入项目内置词库

适用于直接使用项目自带的词库文件。

步骤：

1. 启动程序并打开浏览器页面
2. 选择当前词库，例如“高中词库”或“大学四级词库”
3. 点击“导入内置词库”

系统会把当前项目中的对应 CSV 文件导入到当前选中的词库表中。

### 方式 2：导入自定义 CSV 文件

适用于你自己准备的词库文件。

步骤：

1. 启动程序并打开浏览器页面
2. 选择目标词库
3. 在“上传 CSV”中选择你自己的词库文件
4. 点击“导入上传文件”

CSV 至少需要包含以下表头：

- english
- chinese

示例：

```csv
english,chinese
ability,能力
abandon,放弃
achieve,实现
```

## 开始测试

步骤：

1. 确保当前词库已经导入数据
2. 在测试数量输入框中填写本轮想测试的单词数量
3. 点击“开始测试”
4. 程序会按当前词库抽取单词开始测试
5. 页面会显示当前单词
6. 点击“认识”或“不认识”提交答案

答题时的行为规则：

- 点击“认识”：该单词记为一次正确测试，并根据复习策略安排下次复习时间
- 点击“不认识”：该单词进入错题本，并更快进入下次复习

## 艾宾浩斯式复习规则

当前实现采用分阶段复习间隔：

- 第一次答对后，安排较近的下次复习
- 连续答对越多，下次复习间隔越长
- 答错后，单词会更快重新进入待复习状态

程序会自动记录：

- 首次测试时间
- 最近一次测试时间
- 下次复习时间
- 正确次数
- 错误次数
- 掌握状态

## 错题本

页面右侧会直接显示当前词库的历史错题记录。

每条错题记录会显示：

- 单词
- 中文释义
- 错误次数
- 最近一次答错时间

每轮测试过程中，本轮错题会显示在页面中，也可以点击“导出错题”导出成文本文件。

## 统计面板

页面右侧会固定显示统计区域，不需要额外弹窗。

当前统计窗口会显示：

- 当前词库名称
- 总词数
- 已测试词数
- 待复习词数
- 已掌握词数
- 学习中词数
- 复习中词数
- 累计练习次数
- 累计答错次数
- 错题本条目数
- 本轮已练习数量
- 本轮错题数量

此外还会显示：

- 最近首次测试记录

这个子区域用于查看最近有哪些词是第一次被测试，以及对应时间。

## 导出词库

如果你希望把当前数据库中的词库导出成 CSV：

1. 选择目标词库
2. 点击“导出当前词库”
3. 浏览器会下载导出的 CSV 文件

导出的内容包括：

- english
- chinese
- practice_times
- first_test_date
- last_practice_date
- next_review_date
- correct_times
- wrong_times
- mastery_level

## 清空词库

如果你要清空当前词库：

1. 选择目标词库
2. 点击“清空当前词库”
3. 在浏览器确认提示中确认操作

这会删除当前词库中的所有单词，以及该词库对应的错题记录。
